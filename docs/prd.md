## 1. 产品概述

### 1.1 产品名称
ClearCut-AI (清剪)

### 1.2 一句话描述
面向中文口播视频创作者的一键智能剪辑桌面应用 — 编辑文字就是编辑视频。

### 1.3 目标用户
| 用户画像 | 描述 | 痛点 |
|----------|------|------|
| 知识博主 | 录制教程/课程的创作者 | 大量口误、停顿需要手动剪辑 |
| Vlog 创作者 | 录制口播/Vlog 内容 | 剪辑耗时是录制的 3-5 倍 |
| 播客录制者 | 录制播客/访谈节目 | 多人对话难以高效剪辑 |
| 自媒体团队 | 批量生产短视频内容 | 需要快速粗剪提升效率 |

### 1.4 核心价值主张
1. **省时间**: AI 自动完成 70-80% 的粗剪工作，节省 5-10 倍剪辑时间
2. **零门槛**: 编辑文字就是编辑视频，无需学习专业剪辑软件
3. **中文优化**: 使用最佳中文 ASR 模型，精准识别中文语气词和口误
4. **隐私安全**: 完全本地处理，视频不上传到任何云端
5. **一站式字幕**: 自动生成精准字幕，多种样式一键应用

## 2. 功能需求

### 2.1 P0 核心功能 (MVP)

#### F1: 视频导入
- 支持 MP4、MOV、AVI、MKV 等主流格式
- 显示视频基本信息（时长、分辨率、帧率、大小）
- 拖拽导入或文件选择器

#### F2: 智能转录
- 自动提取音频并转录为文本
- 显示实时处理进度
- 支持中文（FunASR Paraformer）和英文（faster-whisper）
- 词级时间戳对齐（stable-ts）

#### F3: 文本编辑视频
- 文本编辑器与视频预览联动
- 点击文字跳转到对应视频位置
- 选中文字删除 = 删除对应视频片段
- 已删除的文字灰色显示，可恢复

#### F4: 自动静音去除
- 一键检测并标记所有静音段
- 可调节静音阈值和最短静音时长
- 静音段在文本中高亮显示
- 支持全部删除/逐个确认

#### F5: 视频预览
- 实时预览剪辑效果
- 播放/暂停/快进/后退
- 波形时间线可视化
- 已删除段落在时间线中标记

#### F6: 导出
- 导出为 MP4（可选分辨率/码率）
- 导出 SRT 字幕文件
- 显示导出进度

#### F7: 基础字幕
- 从转录结果自动生成 SRT/ASS 字幕文件
- 3 个预制模板：经典白字、经典黄字、卡拉OK
- 基础样式调整：字体、字号、颜色、描边、位置
- 可选择将字幕烧录到视频（硬字幕）或独立导出（软字幕）

### 2.2 P1 增强功能

#### F8: 填充词检测
- 自动检测中文语气词（嗯、啊、那个、就是、然后...）
- 填充词高亮标记，一键全部删除
- 可自定义填充词列表

#### F9: 重复段检测
- 检测说错重来的重复段落
- 显示所有 take，用户选择保留最佳
- 自动推荐保留最佳 take

#### F10: 专业软件导出
- 导出 Adobe Premiere XML
- 导出 Final Cut Pro FCPXML
- 导出 DaVinci Resolve EDL
- 保留原始素材引用

#### F11: 项目管理
- 项目列表/最近项目
- 项目保存/恢复编辑状态
- 项目删除/归档

#### F12: 字幕样式增强
- 渐入渐出动画（ASS `\fad` 标签）
- 卡拉OK 逐词高亮精细调整
- 字幕位置精确调整（像素级定位）
- 实时预览字幕效果（叠加在视频上）
- 字幕与编辑同步（编辑后自动重新对齐时间线）

### 2.3 P2 高级功能

#### F13: LLM 智能分析
- AI 分析内容质量，标记表达不佳的段落
- 智能建议：哪些内容可以精简
- 需要用户配置 API Key

#### F14: TikTok 风格动画字幕
- 集成 pycaps 渲染引擎
- 8+ 预制动画模板：
  - TikTok 弹出 (pop) — 逐词弹出，大字加粗
  - TikTok 高亮 (word_highlight) — 当前词变色
  - Shorts 弹跳 (bounce) — 弹跳出现
  - 极简滑入 (slide) — 从左滑入
  - 打字机 (typewriter) — 逐字打出
- 自定义模板系统（用户创建/导入/分享）
- CSS 样式编辑器（高级用户）

#### F15: 智能缩放
- 人脸检测自动裁剪
- 强调重点时自动放大
- 动态缩放效果

### 3.3 字幕模板规格

**经典白字** (classic-white)
- 字体: Plus Jakarta Sans, 22px, Bold
- 颜色: 白色 (#FFFFFF)
- 描边: 黑色 2px
- 阴影: 黑色 1px 偏移
- 位置: 底部居中
- 动画: 无
- 引擎: ASS

**卡拉OK** (karaoke-glow)
- 字体: Plus Jakarta Sans, 24px, Bold
- 颜色: 白色 (#FFFFFF) → 青色 (#22D3EE) 逐词变色
- 描边: 深蓝 2px
- 位置: 底部居中
- 动画: `\k` 标签逐词高亮，已读过的词变为高亮色
- 引擎: ASS

**TikTok 弹出** (tiktok-pop)
- 字体: Plus Jakarta Sans, 32px, Extra Bold
- 颜色: 白色 (#FFFFFF)
- 描边: 黑色 3px
- 位置: 屏幕中央偏下
- 动画: 每个词缩放弹出 (scale 0→110%→100%)，200ms
- 高亮: 当前词变为品牌红 (#E11D48)
- 引擎: pycaps

## 5. 用户体验流程

### 5.1 首次使用流程
```
打开应用 → 欢迎页(简介) → 导入第一个视频 → 自动处理 → 编辑 → 导出
```

### 5.2 常规使用流程
```
打开应用 → 选择已有项目 / 导入新视频
  ↓
自动处理 (提取音频 → VAD → 转录 → 分析)
  ↓ 显示实时进度
编辑器页面
  ├ 查看转录文本，点击跳转
  ├ 一键去静音
  ├ 一键去废话 (可选)
  ├ AI 分析 (可选)
  ├ 手动微调 (删除/恢复段落)
  ├ 预览剪辑效果
  └ 配置字幕样式 (可选)
  ↓
导出 → 选择格式 + 字幕选项 → 输出文件
```

### 5.3 字幕工作流
```
转录完成 → 字幕自动生成
  ↓
编辑器中实时预览字幕 (叠加在视频画面上)
  ↓
点击 [字幕设置] 按钮
  ├ 选择预制模板 (经典白字/卡拉OK/TikTok弹出...)
  ├ 微调样式 (字体/颜色/大小/位置)
  ├ 选择动画效果 (无/渐入渐出/卡拉OK/逐词弹出)
  └ 实时预览调整效果
  ↓
导出时选择字幕方式
  ├ 软字幕: 导出独立 SRT/ASS 文件
  └ 硬字幕: 烧录到视频 MP4 中
```

## 6. 非功能需求

### 6.1 性能
- 10 分钟视频转录时间 < 1 分钟
- 应用冷启动 < 3 秒
- 编辑操作响应 < 100ms
- 字幕生成 < 5 秒 (10 分钟视频)
- ASS 字幕烧录速度 > 5x 实时

### 6.2 兼容性
- macOS 12+
- Windows 10+ (64-bit)
- 最低 8GB RAM
- 推荐 16GB RAM + GPU (可选加速)

### 6.3 安全
- 所有处理在本地完成
- 不收集用户数据
- LLM API Key 加密存储

### 6.4 商用合规
- 全部依赖使用 MIT/BSD/Apache 许可证
- FunASR 模型在"关于"页面注明出处
- FFmpeg 通过 CLI 调用（独立进程），LGPL 合规
- 应用内包含"开源许可"页面列出所有依赖版权声明
